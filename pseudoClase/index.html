Pseudoclases

-Uno

:active

Selecciona un elemento cuando el usuario lo activa

(por ejemplo, con un clic).

-Tres

:blank

Selecciona elementos cuyo valor de entrada esta vació.

-Cuatro

:checked

Representa cualquier radio, checkbox, u option en un elemento que está marcado o conmutado a un estado.

-Cinco

:current

Representa un elemento o el ancestro de un elemento que se muestra actualmente.

-Seis

:default

Representa cualquier elemento de formulario que sea el predeterminado entre un grupo de elementos relacionados.

:focus-visible

-Diecisiete

:future

Selecciona los elementos que van después del elemento en curso.

-Doieciocho

:hover

Intenta pasar el mouse sobre este enlace y cambiara de color.

-Diecinueve

:indeterminate

Representa cualquier elemento de formulario cuyo estado sea indeterminado.

-Veinte

:in-range

Representa un elemento < input > cuyo valor actual se encuentra dentro de los límites de rango especificados por los atributos min y max.

    Los valores entre 1min y 10max son válidos.

-Veintiuno

:invalid

Selecciona un elemento, como por ejemplo una etiqueta < input >, cuyo estado es no válido.



-Veintidos

:lang

Es utilizada para modificar elementos en función del idioma en el que se determina que están.

Esta cita en inglés tiene una cita anidada adentro.
Esta cita en francés tiene una cita anidada adentro.
Esta cita en aleman tiene una cita anidada adentro.

-Veintitres

:last-child

Representa el último elemento entre un grupo de elementos hermanos.

Este texto no está seleccionado.

¡Este texto está seleccionado!

-Veinticuatro

:last-of-type

Selecciona el último de entre los elementos hermanos de un tipo determinado.

-Veinticinco

:left

Representa todas las páginas de la izquierda de un documento impreso, utilizada con la regla @page at.

-Veintiseis

:link

Selecciona los enlaces no visitados.

Este es un enlace ordinario.

-Veintisiete

:local-link

Selecciona los enlaces que dirigen a páginas que se encuentran en la misma página web que el documento activo.

This is a link on the current page.
This is an external link

-Veintiocho

:is()

Permite construir rápidamente conjuntos de selectores similares estableciendo grupos desde los que cualquier elemento incluido coincidirá.

-Veintinueve

:not

Selecciona elementos que otros selectores no han seleccionado antes y que se han pasado como valor de este selector.

-Treinta

:nth-child

Se especifica con un único argumento, que representa el patrón para los elementos coincidentes.

Los hijos 1, 3, 5 y 7 son seleccionados.

-Treinta y uno

:nth-of-type

Selecciona uno o más elementos de un tipo dado, en función de su posición entre un grupo de hermanos.

3er párrafo.

4to párrafo.

-Treinta y dos

:nth-last-child

Selecciona uno o más elementos en función de su posición entre un grupo de hermanos, contando desde el final.

Quinta línea

-Cuarenta
